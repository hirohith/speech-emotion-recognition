# Speech Emotion Recognition System

A machine learning project that detects human emotions from speech audio using MFCC feature extraction and Random Forest classification.

## Features

- Emotion Detection from WAV files
- MFCC Feature Extraction
- Random Forest Model
- Streamlit Web Application
- Confidence Score Display

## Supported Emotions

- Happy
- Sad
- Angry
- Fearful
- Neutral

## Technologies Used

- Python
- Librosa
- NumPy
- Scikit-Learn
- Streamlit

## Dataset

RAVDESS Emotional Speech Audio Dataset
